如何使用免费 Radmin VPN 软件设置直接连接

  1. 在本地计算机上下载并安装 Radmin VPN。

  2. 创建网络：

    • 按“创建网络”按钮。

    • 设置网络名称和密码。

    • 按“创建”按钮。

    • 新网络将出现在 Radmin VPN 主窗口中。

  3. 在远程计算机上下载并安装 Radmin VPN。

  4. 启动软件并按“加入网络”按钮。

    • 在对话框中输入网络名称和密码。

    • 按“加入”按钮。

    • 先前创建的网络及其节点将显示在 Radmin VPN 主窗口中。

已成功建立电脑之间的连接。您可以从远程电脑的右键菜单中调用“ping”命令来检查连接。

如何通过互联网控制远程计算机

  1. 在远程计算机上运行 Radmin VPN，在主菜单中找到“帮助 -> 安装 Radmin Server”。

  2. 在 Radmin Server 设置中添加用户并设置用户权限。

  3. 使用 Radmin VPN 连接本地计算机和远程计算机。

  4. 选择：远程计算机的快捷菜单中的“Radmin-- > 完全控制”。

  5. 输入设置 Radmin Server 时设置的用户名称和密码。

为什么我不能在游戏网络上看到游戏服务器？

或许现在只有你一个人在玩这个游戏。稍后再尝试连接或邀请朋友加入。

我不能连接到远程电脑

检查本地和远程电脑上的防火墙和防病毒设置。Radmin VPN和游戏不应被杀毒软件或防火墙阻止（此外，你可以禁用杀毒软件或防火墙）。

为什么连接是中继/TCP？

当无法建立直接连接时，将建立中继/TCP连接。尝试加入一个游戏网络。如果与游戏网络参与者的大部分连接都是中继/TCP，那么原因很可能在您的网络提供商那里。从您的提供商那里获取公共IP地址或通过另一个网络提供商连接可能会有帮助。

我无法删除Radmin VPN

如果你在删除Radmin VPN时遇到问题，使用微软实用程序。

  1. 下载实用程序并运行。
  2. 点击“下一步”，选择“卸载”。
  3. 选择Radmin VPN程序，点击“下一步”。
  4. 等待实用程序完成工作并重启电脑。
