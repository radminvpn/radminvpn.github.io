接続

無料ソフトウェアのRadmin VPNを使用したダイレクト接続の設定方法

  1. ローカルコンピューターにRadmin VPNをダウンロードしてインストールします。

  2. ネットワークを作成します：

    • [ネットワークを作成]ボタンを押します。

    • ネットワーク名とパスワードを設定します。

    • [作成]ボタンを押します。

    • Radmin VPNのメインウィンドウに新しいネットワークが表示されます。

  3. リモートコンピューターにRadmin VPNをダウンロードしてインストールします。

  4. ソフトウェアを起動して[ネットワークに参加]ボタンを押します。

    • ダイアログボックスにネットワーク名とパスワードを入力します。

    • [参加]ボタンを押します。

    • 先ほど作成されたネットワークとそのノードがRadmin VPNのメインウィンドウに表示されます。

コンピューター間のコネクションが確立されました。接続を確認するには、リモートコンピュータのコンテキストメニューから「ping」コマンドを呼び出します。

インターネット上でのリモートコンピューターのコントロール方法

  1. リモートコンピューターでRadmin VPNを実行し、メインメニューから[ヘルプｰ>Radmin Serverをインストールします]を探してください。

  2. Radmin Serverの設定で、ユーザーの追加をし、ユーザー権限を設定します。

  3. Radmin VPNを使ってローカルコンピューターとリモートコンピューターを接続します。

  4. 以下を選択します：リモートコンピューターのショートカットメニューの[Radmin-- > フルコントロール]。

  5. Radmin Serverの設定で入力したユーザー名とパスワードを入力します。

Radmin VPNでのプレイ方法

既存のすべての手順については、このリンクをご覧ください。

ゲームが一覧にない場合は、ハマチの指示をお試しください。これで解決することが多いです。

友人とプレイするには、次をご確認ください。

  1. みんなが同じRadmin VPNネットワーク内にいること

  2. みんなが同じゲームバージョンであること

  3. ホスト側のプレイヤーが、ファイアウォールの例外にゲームを追加していること (またはファイアウォールをオフ)。

ゲームネットワーク上にゲームサーバーが表示されないのはなぜですか？

現在このゲームをしているのはあなただけかもしれません。後で接続するか、友人を招待してご参加ください。

リモートPCに接続できません

ローカルおよびリモートPCのファイアウォールおよびアンチウイルス設定をご確認ください。Radmin VPNおよびゲームは、アンチウイルスやファイアウォール (無効にすることも可) でブロックされないようにしてください。

接続がリレー/TCPである理由は？

直接接続ができない場合、リレー/TCP接続が確立されます。ゲームネットワークの1つに参加してみてください。ゲームネットワークの参加者への接続のほとんどがリレー/TCPである場合、この原因はお使いのインターネットプロバイダー側にある可能性が高くなります。プロバイダからパブリックIPアドレスを取得するか、別のインターネットプロバイダ経由で接続すると解決するかもしれません。

インストール時に1603エラー、またはインストールが中断/クラッシュします

この場合は、Radmin VPNをアンインストール後パソコンを再起動してアンチウイルスを無効にし、Radmin VPNをインストールします。これでも解決しない場合はチケットを作成してください。

Radmin VPNに 「Service not started (サービスが開始していません)」 ステータスが表示されます

Radmin VPNの丸いボタンをクリックして、サービスを開始してください。

Radmin VPNに「Waiting for response from the network adapter (ネットワークアダプタからの応答待ち)」ステータスが表示されます。

この場合は、Radmin VPNをアンインストール後パソコンを再起動してアンチウイルスを無効にし、Radmin VPNをインストールします。これでも解決しない場合はチケットを作成してください。

Radmin VPNを削除できません。

Radmin VPNの削除に問題がある場合は、Microsoftユーティリティをお使いください。

  1. ユーティリティをダウンロードして実行します。
  2. [次へ] をクリックし、[アンインストール] を選択します。
  3. Radmin VPNプログラムを選択し、[Next] をクリックします。
  4. ユーティリティが終了するのを待ってからパソコンを再起動してください。
